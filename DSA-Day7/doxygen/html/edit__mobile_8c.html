int EditMobile ( )

Edits the stock quantity of a mobile phone in the store.

Prompts the user to enter the name of the mobile phone for which they want to update stocks. Searches for the mobile phone in the linked list and prints its details if found. Allows the user to enter the updated quantity for the mobile phone. Validates the input and updates the quantity if valid.

Returns
int Returns SUCCESS if the update is successful, otherwise FAILURE.
